HTML5

O HTML5 é a nova versão do HTML4 e um dos seus principais objetivos é facilitar a manipulação dos elementos, possibilitando o developer modificar as características dos objetos de forma não intrusiva, fazendo com que isso fique transparente para o utilizador final.

CSS

Cascading Style Sheet(CSS) é utilizado para definir o estilo nas páginas web que contêm elementos HTML. Define o background color, tamanho da letra, etc...

Elements Foi usado Descrição
Universal done Aplica os estilos a todos os elementos da página
Tipo done Usa o nome do marcador a estilizar
Classe done Aplica o estilo a uma determinada classe
Id done Permite identicar um elemento da página
Atributo clear Aplica o estilo a um atributo do elemento
Pseudo-Classe done Seletor associado ao estado de um elemento
Pseudo-Elemento done Aplica o estilo a parte dos elementos


Elements Foi usado Descrição
Descendentes done div p – todos os p que estão “dentro” de um div
Filhos clear div > p – todos os p que estão “logo dentro” de div
Adjacentes done div ~ p – todos os p que estão colocados depois de um div
Imediatamente Adjacentes done div + p – todos os p que estão colocados “imediatamente” depois de um div
Agrupação done Os elementos ficam com o estilo igual, sendo que estão a ser utilizados nas páginas principais
